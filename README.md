# restaurant-recommendation
Restaurant Recommendation System using Machine Learning that suggests restaurants based on user cuisine preferences. The project uses TF-IDF Vectorization and Cosine Similarity for content-based recommendations and includes a Streamlit web application for interactive user experience.

## Project Description

Restaurant Recommendation System is a Machine Learning project that recommends restaurants to users based on their preferred cuisine choices.

## The Problem Statement

Users often struggle to find restaurants matching their food preferences from large restaurant datasets.

---

## The Solution

This project uses a content-based filtering recommendation system to suggest restaurants based on cuisines entered by the user.
